# Introduction

Affinity CSS

  • Based on Vercel's Design System
  • Some Tailwind's Utility Classes
  • Flex Layout Implementation
  • Sass / Scss mixins for Custom
  • Light / Dark Built-In
  • Under-Development

Installation via yarn or npm

yarn add affinity-css
# or 
npm i affinity-css

Use via cdn

<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/affinity-css@latest/css/index.css">

Minified Version

<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/affinity-css@latest/css/index.min.css">

# Theming

affinity-css supports Light/Dark Theme.

Add the Attribute data-theme : theme-dark for Dark Mode.

Add this script for toggling.

const html = document.querySelector('html');
html.dataset.theme = `theme-dark`;
let isDark = true;
function switchTheme() {
  isDark = !isDark;
  if (isDark) {
    html.dataset.theme = `theme-dark`;
  } else {
    html.dataset.theme = `theme-light`;
  }
}
